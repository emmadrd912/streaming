# Site Streaming
## Architecture du site
### For a premium user
```
Home
-- Login
--- Forgotten password
--- Catalog (with all movies and series)
---- Player
--- My account
---- Edit
--- Billing
---- Add Payment method
---- Cancel plan
--- Invoices
--- Agenda
--- Logout
```
### For a free user
```
Home
-- Login
--- Forgotten password
--- Catalog (with 2 random movies and 1 random tv show episode per day)
---- Player
--- My account
---- Edit
--- Billing
---- Add Payment method
---- Cancel plan
--- Invoices
--- Agenda
--- Logout
```
### For an admin user
```
Home
-- Login
--- Forgotten password
--- Catalog (with all movies and series)
---- Player
--- My account
---- Edit
--- Admin panel
---- Create user
---- Edit 
---- Delete
--- Video panel
---- Create content
---- Edit
---- Delete
--- Agenda panel
---- Create agenda
--- Agenda
```
